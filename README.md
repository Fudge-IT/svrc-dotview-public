# SVRC DotView Public Releases

This repository hosts public release assets for **SVRC DotView**.
Source code is maintained in a separate private repository.

## Download

1. Open the [Releases](https://github.com/Fudge-IT/svrc-dotview-public/releases) page.
2. Download the latest `SVRCDotView-Setup-win-x64.exe` for Windows.
3. Run the installer.

## Quick Start (Windows)

1. Open `SVRC DotView`.
2. Click `Open Folder` and choose a folder containing `.vim`, `.svg`, `.brf`, or `.brg` files.
3. Click a file in the left list to preview it.

## Favorites

1. Open a folder.
2. Click `Favorites` to expand.
3. Click `Add` to save the current folder.
4. Later, choose that folder from the favorites list to open it quickly.
5. Use `Remove` to delete a saved favorite.

## Search

1. Type in `Search files...` to filter files in the opened folder tree.
2. Results update automatically as you type.
3. Click `Clear` to reset search and return to normal folder view.

## Settings (Colors)

1. Click `Settings`.
2. Choose `Dot`, `Background`, and `Accent` colors.
3. Click `Apply`.
4. Your settings are saved and reused next time you open the app.

## Zoom / Fit Behavior

- The preview starts in **auto-fit** mode so images fit the preview area.
- Clicking `-` or `+` switches to **manual zoom**.
- Clicking `Fit` re-enables **auto-fit**.

## Export

Use `Export` to save the currently opened image as:

- `.brg` (Duxbury-friendly graphics import)
- `.brf`
- `.vim` (Duxbury PictureBraille-style)
- `.svg`

## Check Updates

Click `Check Updates` in the app toolbar to look for a newer release.
